# 测试增量更新功能

#测试 #增量更新 #索引

> 这是一个用于测试知识库增量索引更新功能的测试笔记，验证系统是否能正确识别新增笔记并更新索引。

## 测试目标

1. 验证 `.index-state` 文件能正确记录 commit ID
2. 验证增量更新能识别新增的笔记文件
3. 验证增量更新能正确更新索引文件
4. 验证性能优于全量更新

## 测试步骤

1. 创建本笔记
2. 提交到 Git
3. 运行增量更新命令
4. 检查索引是否正确更新

## 预期结果

- 索引文件中应该包含本笔记的信息
- `.index-state` 应该更新到最新的 commit ID
- 标签索引应该包含 "增量更新" 标签
- 分类索引中 Inbox 计数应该增加

- 增量更新脚本: `update-index.py --incremental`
