# 个人笔记知识库 Skill

一个智能的个人知识管理系统，支持多级目录分类、标签管理、Git 同步，完全兼容 Obsidian 查看和编辑。

## 触发条件

| 触发词 | 功能 |
|--------|------|
| `记笔记: <内容>` | 创建新笔记 |
| `知识库状态` | 查看知识库状态和统计 |
| `同步知识库` | 手动触发 Git 同步 |
| `搜索知识库: <关键词>` | 搜索笔记 |
| `整理知识库` | 重建索引 |

## 首次运行：初始化

首次运行时，定位并初始化知识库：

1. **定位知识库**：按以下顺序搜索 `.kb-config.yaml` 文件：
   - 当前工作目录
   - `~/knowledge/`（默认位置）
   - 如果都未找到，询问用户指定路径

2. **如果知识库未找到**，询问用户选择存储位置

3. **克隆或初始化**：
   - 从 https://github.com/jinjimz/knowledge 克隆
   - 如果克隆失败（空仓库），初始化结构

4. **验证配置**：检查 `.kb-config.yaml` 是否存在且有效

## 工作流程：创建笔记

当用户发送 `记笔记: <内容>` 时：

1. **解析输入**：提取标题、标签、分类、附件
2. **智能分类**：匹配现有分类或建议新分类
3. **处理附件**：复制到 `_attachments/YYYY-MM-DD/`
4. **生成笔记**：填充模板，生成 AI 摘要
5. **更新索引**：tags.yaml, categories.yaml, notes-manifest.yaml
6. **Git 本地提交**
7. **检查远程推送**：达到阈值后自动推送

## 重要说明

1. **兼容 Obsidian**：使用标准 Markdown 格式，支持 wiki 链接
2. **双重标签格式**：frontmatter 和行内 `#标签` 格式
3. **Git 安全**：推送前始终先拉取，优雅处理冲突
4. **索引自动维护**：每次创建笔记时自动更新索引
